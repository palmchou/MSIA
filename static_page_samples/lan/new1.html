GDG珠海線下活動 DevFest


真高兴, DevFest 珠海大会能在北师珠 的大力支持下举行!

在此正式 倡议大家都来来参加 ;-) 没来得及报名的也没有关系,现场报名就好!

DevFest 就是开发者节的意思,是 Google 赞助发起的全球性开发者活动,也是 GDG 社区的年度活动. 活动内容完全由世界各地的 GDG 社区志愿者自行策划组织,纯粹的技术分享活动,没有广告式的宣讲和厂商展位,过一个属于开发者自己的节日.

## 概述:

Android Wear,Android L,Polymer, Material Design,移动开发,多屏适配... 各种有趣的新技术,经验分享,开发案例,以及可以动手参与的 Codelab ,参加 DevFest 可以学习最新,最酷的技术,和技术专家面对面交流. 参会者还将获得社区T恤,Google 贴纸和神秘礼物,作为一名开发者决不能错过这个免费的技术盛宴!

+ 活动时间: 2014年10月18日(周六) 13:30~18:30

+ 活动地点: 珠海市唐家湾镇 金凤路 北京师范大学珠海分校 励耘楼 a112

+ 公交线路: 乘 69/10A/k3/70/72A 到"北师大"站下

+ 城铁交通: 唐家湾 站对面 ;-)

## 关于

GDG , 即 Google Developer Groups 的缩写(not 郭德纲)或 谷歌开发者社区,是谷歌开发者部门发起的全球性技术社区.

GDG 是面向对 Google 和开源,开放技术感兴趣的公益性开发者社区,内容涵盖 Web/Chrome,Android 和其它 Google API 等.

全国已成立了18个 GDG 社区,赶快加入你身边的 珠海GDG 吧.

注册为 珠海GDG 成员, 不仅为获得特种纪念品 进而, 通过 珠海GDG 可以:

+ 第一时间获知谷歌最新的技术,

+ 可以学到如何去谷歌平台上赚钱的思路和方法,

+ 可以认识很多有可能将来一起走上自己创业道路的人,

+ 可以学会把你的创新带向国际市场,

+ 参加那里的活动有经常和国际上的开发者们进行交流的机会...

+ 这在珠海没有其它社区和会议可以做到!

協會簡介

澳門軟件行業協會是一個非牟利的民間學術團體。英文名稱：Macau Software Industry Association，簡稱：澳門軟件協會，成立於2014年。 協會廣泛團結澳門從事軟件開發、研究、教育、應用、保護等領域的專業人士及企業，通過市場調查、資訊交流、諮詢評估、行業自律、知識產權保護、資質認定、教育培訓、出版發布、政策研究、會議展覽等方面的工作，促進澳門軟件產業的健康發展。
